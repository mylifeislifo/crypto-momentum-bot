# [스킬] 클로드 메타 스킬 생성기 (Skill Generator)

## 1. 발동 조건
- 사용자가 "이 패턴을 스킬로 저장해", "방금 해결한 문제를 문서화해"라고 요청할 때 발동

## 2. 준수 설계 규칙 (레시피)
- **표준 마크다운 형식 강제:** 생성되는 모든 스킬 문서는 `.md` 확장자로 작성하며, 내부 지식 저장소(Vault)에서 검색과 렌더링이 직관적이도록 깔끔한 헤더(`#`, `##`) 구조를 유지한다.
- **필수 포함 항목:** 
  1. `발동 조건 (Trigger)`
  2. `절대 규칙 (Hard Rules)`
  3. `안티 패턴 (Do NOTs - 피해야 할 코딩 방식)`
  4. `지속성 업데이트 (Changelog)`
- **컨텍스트 분리:** 특정 프로젝트에 국한된 변수명이나 하드코딩된 값은 제외하고, 범용적으로 재사용 가능한 아키텍처 수준의 패턴만 추출하여 기록한다.

## 3. 실패 및 보완 (지속성 업데이트)
- [2026.05.22 추가] 스킬 문서 작성 시 모호한 표현("되도록 ~한다", "상황에 맞춰 ~한다")을 절대 금지하고, 기계가 즉시 판별할 수 있는 명확한 조건문 형태로 서술할 것.
