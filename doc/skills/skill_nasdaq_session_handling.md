# [스킬] 나스닥(NASDAQ) 세션 및 타임존 핸들링 표준

## 1. 발동 조건
- 미국 주식/선물 시장의 데이터를 수집하거나 진입/청산 로직(Signal)을 작성할 때

## 2. 준수 설계 규칙
- **타임존 고정:** 모든 시간 데이터(Timestamp)는 내부적으로 UTC로 저장하되, 조건문이나 로그를 출력할 때는 반드시 `pytz.timezone('US/Eastern')` (EST/EDT)로 변환하여 계산한다.
- **세션 분리:** 로직 내에 정규장(RTH: 09:30~16:00 EST)과 시간외장(ETH: Pre-market & After-hours)을 구분하는 플래그(Flag)를 반드시 구현한다.
- **오프닝 필터:** 별도의 명시가 없는 한, 정규장 개장 직후 15분(09:30~09:45 EST)은 변동성이 극심하므로 신규 진입을 차단하는 하드 리미트를 포함한다.
